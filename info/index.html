画像・動画のダウンロードについて

以下のサイトからダウンロードできます。

動画・画像のダウンロード

サイトにアクセスすると、全ての画像と動画が読み込まれます。通信料などに注意してください。

メッセージについて

メッセージは最新の100件のみが表示されます。

また、メッセージの取り消しをしたい場合はこのページの作成者に申し出てください。

なお、ユーザー名は１２文字までしか入力できません。

それ以上の文字数を入力した場合、最初から１２文字を表示します。

また、なりすましを防ぐため、一部のユーザー名(以降登録ユーザー名)の使用にはパスワードが必要になります。

パスワードを入力し、そのユーザー名を使用している人(以降認証ユーザー)は青色の文字で表示され、また登録ユーザー名を使用する際にパスワードを入力しなかった、もしくは間違えてしまったユーザーは赤色に表示されます。

登録ユーザー名の仕様と使い方について

以前はformでユーザー名のところに「登録ユーザー名/パスワード」と入力していましたが、パスワードのみでも登録ユーザー名を使用することができるようになりました。

自動更新について

右上の「自動更新」チェックボックスにチェックを入れると自動更新が有効になります。

自動更新をオンにしている間は手動での更新が出来ません。

自動更新中に新しいメッセージを受信すると少しの間メッセージが消え、新しいメッセージが表示されます。

それを不便に思う場合は、自動更新をオフにしておくことをお勧めします。

URLの送り方について

URLを送る際は、「https://」もしくは「http://」で始まるようにしてください（省略しないでください）。

また、URLの最後には半角の空白を開けてください。

その空白は削除されます。

そのため、URLの後に空白を開けたい場合は空白を２つ入力します。そうすることで、片方の空白は消えますが、片方は残るのでURLの後に空白を開けることができます。

一般に、URLの後にn文字の空白を開けたい場合は、n+1文字の空白を開けて送信してください。

例

OK：https://sigma0913.github.io/mail/info/にアクセス（空白は消えている。）

NG：sigma0913.github.io（「https://」を省略しているためURLとして認識されていない。）

NG：https://sigma0913.github.io/mail/info/にアクセス（空白がなく、余計な部分がURLとして認識されている。）

NG：https://sigma0913.github.io/mail/info/　にアクセス（全角のスペースを入れている。）

スタンプの出し方について

/stamp/のあとに、以下の文言をつけるとスタンプを出すことができます。

  • good
  • balloon
  • oyster_shell
  • tenkaicon
  • tenkalogo
  • html
  • css
  • javascript

なお他にも隠されたスタンプが存在しています。

このページの開発者ビューからソースをみれば、コメントアウトしてあるスタンプも見ることができます。

また、sigma0913のgithubのリポジトリからでも閲覧が可能です。
